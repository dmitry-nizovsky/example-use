# example-use

**Install project**
```shell
npm install
```

**How to Dev**
```shell
npm run dev
npm run dev:watch // watch mode
```

**How to Build for Production**
```shell
npm run build
```

**How to test**
```shell
npm run test
npm run test:watch // watch mode
```

**How to code style check**
```shell
npm run eslint // check js code
npm run stylelint // style
```
